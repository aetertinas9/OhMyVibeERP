# 충돌 해결 로그

작업 중 발생한 모든 충돌과 그 해결 과정을 기록한다. 최신 항목을 위에 추가한다.

## 항목 템플릿

```
## CONFLICT-NNN: <한 줄 요약>
- 일시: YYYY-MM-DD HH:MM
- 수행 중이던 작업: <어떤 단계/작업 중이었는지, 관련 커밋>
- 충돌 내용: <무엇과 무엇이 충돌했는지, 에러 메시지·충돌 파일 등 구체적으로>
- 원인: <왜 발생했는지>
- 해결: <어떻게 해결했는지. 선택지가 여럿이었다면 왜 이 방법을 골랐는지>
- 결과: <해결을 어떻게 확인했는지(테스트 등)와 해결 커밋 해시>
```
