# Metrics.md — 회차별 정량 지표 (기록 전용)

매 회차 종료 시 기존 기록에서 숫자만 집계해 표에 한 줄을 추가한다.
실증 분석 전용이며, Redteam.md와 마찬가지로 구현에는 개입하지 않는다.

## 집계 기준

- 지시 수: 해당 회차의 `Vibe.md` 항목 수
- 구현 줄 수(+/-): 회차 시작~종료 커밋 구간의 `git diff --shortstat` 합계
- 커밋 수: 해당 회차의 `[단계 N]` 커밋 수
- 버그 수: `docs/log/round-NN_*.log`의 `[BUG-*]` 항목 수
- 충돌 수: 해당 회차에 `docs/conflict-log.md`에 추가된 항목 수
- 레드팀 판정: `Redteam.md` 해당 회차의 판정(충족/부분 충족/미충족)
